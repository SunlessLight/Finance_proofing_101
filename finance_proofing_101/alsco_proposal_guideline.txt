This is where you write an abstract for the event. Be concise. Include a brief description of the event, the objectives of the event and the key details about the event. Also, include the contact information of the ECs (phone number and Sunway iMail address).

The following is an example of what the abstract should look like. Please note that the following is only a sample and should not be copied word-for-word:

The January ‘25 ALSCO Community Service (COMSERVE) Department would like to propose and seek approval for (name of event), which will be held on (date), from (start time) to (end time) at (venue). This event will be open to (target participants), and aims to (main objective). The registration fee will be (price) per person, with an early-bird fee of (price) per person.

(rough outline of event flow) (may span across multiple paragraphs depending on the length of the itinerary)

An Internal Trial Run (ITR) will be conducted on (date) from (start time) to (end time), during which committee members will test the planned activities and refine any logistical or material concerns. This will ensure the committee is fully prepared for the Actual Event (AE). This will be followed by an External Trial Run (ETR) on (date) from (start time) to (end time) with XX ALSTAR volunteers acting as participants to simulate the event experience with an unbiased viewpoint. The feedback taken from this session will be used to finalise the event framework.

All ALSTAR participants are eligible to receive:
 ONE (1) Volunteer Point for attending the External Trial Run (ETR).
ONE (1) Participation Point for attending the Actual Event (AE).

The estimated cost of this event is (price), with funds from the registration fees covering any expenses. Any surplus funds will be either donated to charity in alignment with COMSERVE’s goals or stored in the COMSERVE funds to be used for future events.

We believe this event offers a valuable opportunity for participants to develop skills that are both personally and professionally impactful.
